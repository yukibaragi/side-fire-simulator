# サイドFIRE・フリーランス転向シミュレーター

フリーランスエンジニア＋副業（ライター等）の二本柱で生活する人向けの資産シミュレーター。

**[デモを見る →](https://ibaragiyuuki.github.io/side-fire-simulator/)**

## 特徴

- フリーランス特有の税・社会保険計算（所得税・住民税・国民健康保険・国民年金・消費税）
- サイドFIRE達成の可視化（プログレスリング・資産推移チャート）
- 青色申告控除・小規模企業共済・iDeCo等の節税シミュレーション
- 子どもの教育費（公立/私立）の考慮
- レスポンシブ対応

## サイドFIRE達成条件

```
投資資産 × 4%（年間取り崩し額）+ 副業収入 + 配偶者収入 ≥ 年間総支出
```

## 開発

```bash
npm install
npm run dev
```

## 注意事項

- 国民健康保険料は代表的な料率による概算値です（自治体ごとの料率差は考慮していません）
- 投資リターンは確定ではありません
- 重要な判断の際は税理士・FP等にご相談ください
